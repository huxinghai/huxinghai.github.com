Octopress 添加theme 与 Comments

添加theme步骤如下:
首先到github选择主题

        cd octopress
        $ git clone GIT_URL .themes/THEME_NAME
        $ rake install['THEME_NAME']
        $ rake generate

添加comments步骤如下:
到disqus注册帐户
然后打开”_config.yml”，找到了disqus相关的配置项，修改即可
        disqus_short_name: your_disqus_short_name
        disqus_show_comment_count: true
